Detailed Description

A base class for various motion handlers The handler for the Entities's actuators, which in this case are the two wheel actuators. Its main job is to translate the directional commands from the user into the appropriate differential drive wheel speeds.

MotionBehavior translates velocity and position to a new position. Both of these are straightforward, but the framework allows for more sophisticated models of motion in which each wheel has distinct speed.
