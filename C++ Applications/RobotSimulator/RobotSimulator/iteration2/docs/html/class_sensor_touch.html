Detailed Description

A derived class from sensor base class;a representation of a touch sensor. Touch or "bump" sensors are "active" when pressed. For our purposes, imagine a series of bump sensors around the perimeter of the robot. A collision will activate the sensor at a particular point of contact, which translates to an angle of contact.
