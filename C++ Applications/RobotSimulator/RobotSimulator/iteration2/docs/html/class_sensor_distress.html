Detailed Description

A derived class from sensor base class;a representation of a distress sensor. Touch or "bump" sensors are "active" when this entity is freeze. For our purposes, imagine a series of bump sensors around the perimeter of the robot. A special collision will inactivate the sensor at a particular point of contact and also unfreeze this entity.
