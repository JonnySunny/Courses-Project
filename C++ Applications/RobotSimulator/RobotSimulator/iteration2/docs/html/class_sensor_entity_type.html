Detailed Description

A derived class from sensor base class;a representation of a entity type sensor. Touch or "bump" sensors are "active" when pressed . For our purposes, imagine a series of bump sensors around the perimeter of the robot. A collision or will have collision in a defined range will activate the sensor.
