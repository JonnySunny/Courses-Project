Detailed Description

The handler for the homebase's actuators. HomebaseMotionHandler manages the modification to the velocity based on only collisions. MotionBehavior translates velocity and position to a new position. Both of these are straightforward, but the framework allows for more sophisticated models of motion in which each wheel has distinct speed.
